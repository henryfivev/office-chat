# OfficeChat V2 产品重构 PRD

## 产品定位

OfficeChat 不是聊天室。

OfficeChat 是一个伪装成办公协作工具的实时沟通平台。

用户打开页面时：

应该首先看到：

* 文档
* 知识库
* 协同编辑记录

而不是：

* 聊天窗口
* 消息列表
* 输入框

聊天能力是附着在办公场景上的。

---

# 核心原则

## V1（错误）

```text
聊天室
↓
换主题
↓
飞书
钉钉
腾讯文档
```

本质还是聊天室。

---

## V2（正确）

```text

钉钉知识库
腾讯协同文档

↓

```

本质是文档产品。

# Codex 重构要求

重点不是改 UI。

重点是：

1. 删除聊天室思维
2. 删除消息列表思维
3. 引入 Document Shell
4. 引入 Comment System
5. 三种主题拥有不同布局
6. 三种主题拥有不同评论渲染逻辑
7. 聊天内容全部伪装成评论、编辑记录、讨论内容
8. 老板键改为纯文档模式
9. 图片改为附件模式
10. 在线人数改为协作者概念

最终效果：

用户打开页面第一眼认为：

这是飞书/钉钉/腾讯文档。

而不是：

这是一个聊天室。
